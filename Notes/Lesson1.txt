1) npm install

2) component mimarisi, single page application.

3) ecma script -> geliştirilmiş javascript denebilir.

	Transpiler ile yazdığımız kod javascript'e çevrilir.

4) saf javascript'te, variable tanımlarken "var" keyword'ü kullanılır.
	
	Ancak var kullanmıyoruz, "let" keyword'ünü kullanıyoruz.
	
	var kullanıldığı zaman scope olayı yok oluyor. 
	
	Mesela global bir değişkenin var, ve altta if bloğu var mesela. Eğer var kullanırsan
	globalde olan değişken de değişiyor ve patlıyoruz.
	
5) javascript type safe değildir. Yani bir değişken tanımladık, int türünde daha sonra
aynı değişkeni string bir değerde tekrardan tanımlayabiliriz.

6) const -> constant, const olarak bir değişkene, bir değer atayınca buna başka bir şey
atayamazsın. Read Only olarak geçer.
